# Tauri + Solid + Typescript

Shortcuts is app for starting tools on disk

### Start

```bash
cargo run ./main.rs
```

### Start

```bash
yarn tauri dev
```

### Rust Update

```bash
cd ./src-tauri
cargo update
```

#### Default export

```bash
tauri build
```
